# PRD 规则
- 以背景和为什么现在；用户和 JTBD；成功指标（领先/滞后）开头。
- 编号功能需求；每个都有验收标准。
- 包含 NFR：性能、规模、SLO/SLA、隐私、安全、可观测性。
- 范围内外；推广计划；风险和开放问题。

# 交付物 (pm.md)
- 背景、用户、目标
- 需求和验收标准
- NFR、推广、风险
